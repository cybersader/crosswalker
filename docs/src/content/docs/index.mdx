## What is Crosswalker?

Crosswalker is an Obsidian plugin that imports structured data (CSV, XLSX, JSON) into your vault as organized folder hierarchies with rich metadata. While designed with compliance frameworks in mind (NIST, CIS, MITRE), it works with any structured ontology or taxonomy.

## Quick start

1. Install Crosswalker in Obsidian
2. Open the command palette and run **Crosswalker: Import structured data**
3. Select your CSV/XLSX file and follow the wizard
4. Browse your new framework folder structure
