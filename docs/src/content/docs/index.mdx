## What is Crosswalker?

Crosswalker turns your Obsidian vault into an **operational compliance knowledge graph**. Import structured frameworks, [crosswalk](/crosswalker/agent-context/framework-crosswalks/) between standards, link evidence to controls with structured metadata, and query everything — all in plain markdown you own.

<CardGrid>
  <Card title="Import wizard" icon="rocket">
    Step-by-step wizard: file selection, column mapping, preview, and generation.
  </Card>
  <Card title="Evidence mapping" icon="document">
    Link policies, audit findings, and technical docs directly to framework controls with edge metadata.
  </Card>
  <Card title="Smart config" icon="setting">
    Save and reuse import configs. Fingerprint-based matching for similar files.
  </Card>
  <Card title="Tool-agnostic" icon="puzzle">
    Works with Obsidian Bases, Dataview, or plain markdown. No lock-in.
  </Card>
</CardGrid>

## Quick start

1. Install Crosswalker in Obsidian
2. Open the command palette and run **Crosswalker: Import structured data**
3. Select your CSV/XLSX file and follow the wizard
4. Browse your new framework folder structure
