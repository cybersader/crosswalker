## Role

**Industry group** (financial sector) — creates the CRI Profile, adopted by major financial institutions.

## Key outputs

- [CRI Profile v2.1](/crosswalker/reference/framework-data-sources/#cri-profile-v20) — diagnostic statements, subject tags, multi-sheet crosswalks
- Maps to: NIST CSF, [FFIEC](/crosswalker/reference/registry/ffiec/) CAT, CISA CPG

## In Crosswalker

CRI is a complex import — multiple sheets, tag aggregation, hierarchical forward-fill. See [framework data sources](/crosswalker/reference/framework-data-sources/#cri-profile-v20).
