## Role

**Mapping organization** (submission-based) — accepts crosswalk submissions between NIST frameworks and others. Operated by [NIST](/crosswalker/reference/registry/nist/).

## How it works

Organizations submit mappings using 5 [formal relationship types](/crosswalker/concepts/schema-matching/#nist-olirs-formal-relationship-types): Subset, Intersects, Equal, Superset, Not Related. Each with a 1-10 strength rating.

## The staleness problem

OLIR is submission-based — there's no automatic mechanism to flag stale crosswalks when either framework updates. See [crosswalk staleness](/crosswalker/agent-context/framework-crosswalks/#crosswalk-staleness).
