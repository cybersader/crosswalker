## Role

**Regulator** (US financial) — mandates cybersecurity assessment tools and examinations for banks and credit unions.

- [FFIEC CAT](https://www.ffiec.gov/cyberassessmenttool.htm) — Cybersecurity Assessment Tool
- [FFIEC Cybersecurity Resource Guide](https://www.ffiec.gov/press/pdf/FFIECCybersecurityResourceGuide2022ApprovedRev.pdf)
