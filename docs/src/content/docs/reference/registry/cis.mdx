## Role

**Industry group** — creates practical, prioritized security controls and system hardening benchmarks.

## Key outputs

- [CIS Controls v8](/crosswalker/reference/framework-data-sources/#cis-controls-v8) — 18 controls, 153 safeguards, organized by Implementation Group (IG1/IG2/IG3)
- CIS Benchmarks — System-specific hardening guides
- [CIS Controls Navigator](https://www.cisecurity.org/controls/cis-controls-navigator) — Interactive mapping tool

## In Crosswalker

CIS represents the "hard" end of [ontology evolution](/crosswalker/concepts/ontology-evolution/) — `unrestricted` breaking changes, `evolving` IDs, `prose_only` changelog, `radical` hierarchy changes. [EvolutionPattern](/crosswalker/agent-context/zz-log/2026-04-03-evolution-pattern-taxonomy-draft/) recommends SCD Type 1 (simple re-import).
