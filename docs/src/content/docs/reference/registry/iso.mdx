## Role

**Standard body** (international) — creates globally recognized information security management standards.

## Key outputs

- ISO 27001 — Information Security Management System (ISMS) requirements
- ISO 27002 — Security controls guidance
- ISO 31000 — Risk management

## In Crosswalker

ISO standards are commonly mapped to via crosswalks from [NIST](/crosswalker/reference/registry/nist/) and [CIS](/crosswalker/reference/registry/cis/) frameworks.
