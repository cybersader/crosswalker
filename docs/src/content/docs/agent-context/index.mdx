This section contains internal project knowledge useful for development, AI-assisted coding, and exploration. It includes design decisions, prior art research, open questions, and an exploration log.

Unlike the rest of the docs, this section is written for **contributors and AI agents** rather than end users. It provides the "why" behind design choices and captures institutional knowledge that would otherwise be lost between sessions.

## What's here

- **[Prior art](/crosswalker/agent-context/prior-art/)** — Existing tools, research, and the history of this project
- **[Vision](/crosswalker/agent-context/vision/)** — Short and long-term goals, design principles
- **[Decisions](/crosswalker/agent-context/decisions/)** — Key technical decisions and their rationale
- **[Open questions](/crosswalker/agent-context/open-questions/)** — Unresolved design questions
- **[Tradeoffs](/crosswalker/agent-context/tradeoffs/)** — Known tradeoffs and their justifications
- **[Framework landscape](/crosswalker/agent-context/framework-landscape/)** — Overview of compliance frameworks and their data structures
- **[Exploration log](/crosswalker/agent-context/exploration-log/)** — Running notes from development and research sessions

## For AI agents

If you're an AI coding assistant working on this project, start here to understand context that isn't obvious from the code alone. The `.claude/` folder in the repo contains the source versions of these documents with agent-specific instructions.
