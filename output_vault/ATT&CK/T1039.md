---
title: Data from Network Shared Drive
desc: Adversaries may search network shares on computers they have compromised to
  find files of interest. Sensitive data can be collected from remote systems via
  shared network drives (host shared directory, network file server, etc.) that are
  accessible from the current system prior to Exfiltration. Interactive command shells
  may be in use, and common functionality within [cmd](https://attack.mitre.org/software/S0106)
  may be used to gather information.
