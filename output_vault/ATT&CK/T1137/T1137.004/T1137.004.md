title: 'Office Application Startup: Outlook Home Page'
desc: "|-\n  Adversaries may abuse Microsoft Outlook's Home Page feature to obtain\
  \ persistence on a compromised system. Outlook Home Page is a legacy feature used\
  \ to customize the presentation of Outlook folders. This feature allows for an internal\
  \ or external URL to be loaded and presented whenever a folder is opened. A malicious\
  \ HTML page can be crafted that will execute code when loaded by Outlook Home Page.(Citation:\
  \ SensePost Outlook Home Page)\n  \n  Once malicious home pages have been added\
  \ to the user's mailbox, they will be loaded when Outlook is started. Malicious\
  \ Home Pages will execute when the right Outlook folder is loaded/reloaded.(Citation:\
  \ SensePost Outlook Home Page)"
