---
name: De-identification | Release
text: Remove personally identifiable information elements from a dataset prior to
  its release if those elements in the dataset do not need to be part of the data
  release.
discussion: Prior to releasing a dataset, a data custodian considers the intended
  uses of the dataset and determines if it is necessary to release personally identifiable
  information. If the personally identifiable information is not necessary, the information
  can be removed using de-identification techniques.
