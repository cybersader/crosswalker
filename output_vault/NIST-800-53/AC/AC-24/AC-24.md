---
name: Access Control Decisions
text: '[Selection: Establish procedures; Implement mechanisms] to ensure [Assignment:
  organization-defined access control decisions] are applied to each access request
  prior to access enforcement.'
discussion: Access control decisions (also known as authorization decisions) occur
  when authorization information is applied to specific accesses. In contrast, access
  enforcement occurs when systems enforce access control decisions. While it is common
  to have access control decisions and access enforcement implemented by the same
  entity, it is not required, and it is not always an optimal implementation choice.
  For some architectures and distributed systems, different entities may make access
  control decisions and enforce access.
