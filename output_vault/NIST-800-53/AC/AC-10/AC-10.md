name: Concurrent Session Control
text: 'Limit the number of concurrent sessions for each [Assignment: organization-defined
  account and/or account type] to [Assignment: organization-defined number].'
discussion: Organizations may define the maximum number of concurrent sessions for
  system accounts globally, by account type, by account, or any combination thereof.
  For example, organizations may limit the number of concurrent sessions for system
  administrators or other individuals working in particularly sensitive domains or
  mission-critical applications. Concurrent session control addresses concurrent sessions
  for system accounts. It does not, however, address concurrent sessions by single
  users via multiple system accounts.
