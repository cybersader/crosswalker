---
name: Transmission of Security and Privacy Attributes | Anti-spoofing Mechanisms
text: Implement anti-spoofing mechanisms to prevent adversaries from falsifying the
  security attributes indicating the successful application of the security process.
discussion: Some attack vectors operate by altering the security attributes of an
  information system to intentionally and maliciously implement an insufficient level
  of security within the system. The alteration of attributes leads organizations
  to believe that a greater number of security functions are in place and operational
  than have actually been implemented.
